# PREMIUM MOTIVATIONAL REEL ENGINE

## PROJECT

Create a premium, reusable motivational reel template for Instagram Reels, YouTube Shorts, TikTok, Facebook Reels, and YouTube Shorts.

The template must be fully editable so that only the quote, author, colors, and logo need to be changed to generate a completely new reel.

---

## OUTPUT

* Vertical 1080 × 1920 (9:16)
* 30 FPS
* 15–20 seconds
* Mobile-first design
* Professional social-media quality
* Export-ready

---
